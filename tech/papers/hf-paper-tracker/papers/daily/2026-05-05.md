### [T1/T4] MASCing: Configurable Mixture-of-Experts Behavior via Activation Steering Masks
- URL: https://arxiv.org/abs/2604.27818
- 重要度: 🔵Medium
- 要約:
  1. [課題] MoEは疎活性化で推論効率を得る一方、ルーティング依存で安全性や振る舞い制御が難しい。
  2. [手法] ルーティングlogitから挙動を予測する代理モデルを学習し、推論時にステアリングマスクで専門家選択を上書きする。
  3. [結果] 7つの公開MoEでほぼ無視できる追加コストのまま、安全防御成功率や特定挙動の誘導成功率を大幅に改善した。
- 一言メモ: MoE基盤で「再学習せずに振る舞いだけ変える」発想は、金融向け安全制約や出力ポリシー切替の軽量実装に直結する。

### [T3] Web2BigTable: A Bi-Level Multi-Agent LLM System for Internet-Scale Information Search and Extraction
- URL: https://arxiv.org/abs/2604.27221
- 重要度: ⚡High
- 要約:
  1. [課題] Web検索エージェントは、深い単体調査と多数対象の構造化集約を同時に高品質でこなせない。
  2. [手法] 上位オーケストレータと下位ワーカーからなる二層マルチエージェント構成に、run-verify-reflectループと共有ワークスペースを組み合わせた。
  3. [結果] WideSearchで既存2位を大きく上回るRow F1/Item F1を達成し、深掘り系ベンチでも高精度を示した。
- 一言メモ: 多銘柄・多ソースから表形式で根拠付き集約する設計は、企業比較やM&A候補スクリーニングの土台としてそのまま参考になる。

### [T3] From Skill Text to Skill Structure: The Scheduling-Structural-Logical Representation for Agent Skills
- URL: https://arxiv.org/abs/2604.24026
- 重要度: 🔵Medium
- 要約:
  1. [課題] SKILL.mdのような自然言語中心のスキル定義は、検索性・再利用性・リスク審査に弱い。

- 一言メモ: MCP連携を含む社内エージェント群のスキルカタログを機械可読化する設計指針として有用。

### [T2] AnalogRetriever: Learning Cross-Modal Representations for Analog Circuit Retrieval
- URL: https://arxiv.org/abs/2604.23195
- 重要度: ⚪Low
- 要約:

### [T4] Themis: Training Robust Multilingual Code Reward Models for Flexible Multi-Criteria Scoring
- URL: https://arxiv.org/abs/2605.00754
- 重要度: 🔵Medium
- 要約:
  1. [課題] コード向け報酬モデルは正しさ偏重で、多基準・多言語の評価軸を十分に扱えていない。
  2. [手法] 5つの評価観点と8言語を跨ぐベンチと35万超の選好対データを整備し、多基準コードRM群を学習した。
  3. [結果] 多言語転移とスケーリング則を示しつつ、柔軟な多基準スコアリングで既存RMの弱点を補った。
- 一言メモ: 構造化出力の品質評価でも、単一正解ではなく「形式遵守・根拠性・安全性」の多基準RM発想が効く。

### [T1/T4] Efficient Training on Multiple Consumer GPUs with RoundPipe
- URL: https://arxiv.org/abs/2604.27085
- 重要度: ⚡High
- 要約:
  1. [課題] consumer GPU群でのLLM微調整は、PCIe帯域と不均一ステージ配置のためパイプライン気泡が大きい。
  2. [手法] GPUを固定ステージではなく無状態ワーカーとして扱い、ラウンドロビンで動的に計算を割り当てるRoundPipeを提案した。
  3. [結果] 8x RTX 4090で既存法比1.48〜2.16倍の高速化を示し、Qwen3-235BのLoRA微調整も単一サーバで可能にした。
- 一言メモ: PCIe前提・量子化必須の自プロジェクト条件と極めて近く、LoRA/QLoRA訓練基盤の現実解として最重要級。

- 要約:
