### [T1] Long Context Pre-Training with Lighthouse Attention
- URL: https://arxiv.org/abs/2605.06554
- 重要度: 🔵Medium
- 要約:
  1. [課題] 超長文脈の事前学習は通常の自己注意の二乗コストが大きなボトルネックになる。
  2. [手法] 学習中だけ系列を階層圧縮して復元する Lighthouse Attention を被せ、終盤でフル attention に戻す二段階学習を提案した。
  3. [結果] 同条件のフル attention 学習より総学習時間を短縮しつつ、復帰後の最終 loss も改善した。
- 一言メモ: 長い社内文書や決算資料を読む基盤モデルの事前学習コストを下げる発想として実務的。

### [T1] BEAM: Binary Expert Activation Masking for Dynamic Routing in MoE
- URL: https://arxiv.org/abs/2605.14438
- 重要度: ⚡High
- 要約:
  1. [課題] 固定 Top-K の MoE ルーティングは不要な expert 発火を生み、推論レイテンシを押し上げる。
  2. [手法] 学習可能な二値マスクで token ごとに expert sparsity を動的制御し、CUDA kernel と vLLM 統合まで実装した。
  3. [結果] 元性能の98%以上を維持しつつ MoE FLOPs を最大85%削減し、最大2.5倍の decoding 高速化を報告した。
- 一言メモ: PCIe 前提の MoE 推論を直撃するので、現行インフラ制約に最も近い改善候補。

### [T1] Forcing-KV: Hybrid KV Cache Compression for Efficient Autoregressive Video Diffusion Models
- URL: https://arxiv.org/abs/2605.09681
- 重要度: 🔵Medium
- 要約:
  1. [課題] 自己回帰生成では履歴フレームの KV cache が肥大化し、速度とメモリを圧迫する。
  2. [手法] attention head を静的役割と動的役割に分け、前者は構造 pruning、後者は類似度ベース圧縮を行うハイブリッド KV 圧縮を導入した。
  3. [結果] 品質を維持しつつ cache メモリを30%削減し、条件によって 1.35倍から 2.82倍の高速化を達成した。
- 一言メモ: 動画向けだが、head 役割ごとに KV を差別化圧縮する発想は長文 LLM 推論にも転用余地がある。

### [T2] WildTableBench: Benchmarking Multimodal Foundation Models on Table Understanding In the Wild
- URL: https://arxiv.org/abs/2605.01018
- 重要度: 🔵Medium
- 要約:
  1. [課題] 実世界の表画像は複雑なレイアウトと数値推論を含むが、既存評価は整形済み表に偏っていた。
  2. [手法] 自然画像の表402枚と928問からなる in-the-wild 表理解ベンチマークを構築した。
  3. [結果] 21モデル中50%超の精度は1モデルだけで、構造把握と数値推論の弱さが明確に残った。
- 一言メモ: 決算資料や IR スライド中の表を読む金融タスクの難所をそのまま示している。

### [T2] PreScam: A Benchmark for Predicting Scam Progression from Early Conversations

- 要約:

### [T3] RouteProfile: Elucidating the Design Space of LLM Profiles for Routing
- URL: https://arxiv.org/abs/2605.00180
- 重要度: 🔵Medium
- 要約:
  1. [課題] LLM routing 研究は router 本体に偏り、各モデル能力を表す profile 設計が十分整理されていなかった。

### [T3] STALE: Can LLM Agents Know When Their Memories Are No Longer Valid?
- URL: https://arxiv.org/abs/2605.06527
- 重要度: ⚡High
- 要約:
  1. [課題] 長期記憶エージェントは更新後の事実を保持しても、古い前提を無効化して行動を変えるのが苦手である。

- 一言メモ: 顧客属性やポートフォリオ条件が変わる金融対話で、古い前提の残留を防ぐ設計に直結する。

### [T4] Adaptive Teacher Exposure for Self-Distillation in LLM Reasoning
- URL: https://arxiv.org/abs/2605.11458

- 要約:
  1. [課題] 自己蒸留では教師が完全な参照推論を見すぎると、学生能力との差が広がって学習効率を落とす。
